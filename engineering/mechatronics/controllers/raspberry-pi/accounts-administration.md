# Create a user account

- ```sh
  sudo adduser <username>
  ```

<br>
<br>

# Add a user to `sudo` group

- ```sh
  sudo adduser <username> sudo
  ```

<br>
<br>

# Delete a user account

- ```sh
  sudo deluser --remove-home <username>
  ```
