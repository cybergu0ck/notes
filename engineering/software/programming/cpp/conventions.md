## Brace placement

Prefer K&R style over Allmann style.

- The following is K&R style of brace placement for functions, standard C++ libraries use this style.

  ```cpp
  void myFunction(){
      // Code goes here
  }
  ```

- The following is Allmann style of brace placement for functions.

  ```cpp
  void myFunction()
  {
      // Code goes here
  }
  ```
