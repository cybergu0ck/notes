# Debian pacakage managers

<br>
<br>
<br>

## Apt

<br>
<br>

### Update and upgrade

```bash
sudo apt update
```

```bash
sudo apt upgrade
```

<br>
<br>

### Installing a package

```bash
sudo apt install <package>
```

<br>
<br>

### Uninstalling a package

```bash
sudo apt remove <package>
```
