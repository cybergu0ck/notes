# Package management

<br>
<br>
<br>

## apt

<br>
<br>

### Update and upgrade

```bash
sudo apt update
```

```bash
sudo apt upgrade
```

<br>
<br>

### Installing a package

```bash
sudo apt install <package>
```

<br>
<br>

### Uninstalling a package

```bash
sudo apt remove <package>
```

<br>
<br>

## snap

<br>
<br>

### Uninstalling a package

```bash
sudo snap remove <package>
```
