# Emments

| Emment           | Key Binding |
| ---------------- | ----------- |
| Basic template   | !           |
| div element      | .           |
| div with a class | .classname  |
| a tag            | a           |
| image tag        | img         |
| unordered list   | ul          |
| list item        | li          |
| navigation tag   | nav         |

<br>
<br>

- To create script tags,
  ``
  ```
  script[src="<name>"]
  ```
